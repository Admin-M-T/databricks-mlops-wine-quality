# Databricks MLOPs Wine Quality Project

## Overview
This project utilizes DABs,Jobs,MlFlow and Github

## Dataset
UCI Wine Quality Dataset

## Workflow
1. Ingest raw CSV
2. Validate and clean data
3. Split into train and test stes
4. Train classification sets
5. Evaluate model performance
6. Run batch predictions

## Techstack
- Databricks Free Edition
- Databricks Asset Bundles
- Databricks Jobs
- Mlflow
- PySpark
- scikit-learn
- Github
